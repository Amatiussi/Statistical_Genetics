# Genética-Estatística I: Fundamentos

Este repositório contém os materiais da disciplina `Genética-Estatística I (2026)`, ministrada pelo Prof. Dr. Saulo Fabrício da Silva Chaves, da ESALQ/USP, como parte do Programa de Pós-Graduação em Genética e Melhoramento de Plantas. O objetivo deste repositório é disponibilizar os `scripts` e os `exercícios` desenvolvidos ao longo das aulas do semestre em andamento, todos desenvolvidos em R.
